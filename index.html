Scaffolding - automatic code generation framework

Scaffolding is a technique used by many MVC frameworks like ASP.NET MVC and Node.JS etc., to generate code for basic CRUD (create, read, update, and delete) operations against your database effectively. Further you can edit or customize this auto generated code according to your need.
